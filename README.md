# Caesar-Encryption-in-Java
A simple Java project where you can encrypt sentences using the old Caesar encryption method

Simply click "run" and then enter a string.

An example output of the encryption would be:


Enter text to encrypt: 
hello world
Enter shift key (0 - 25): 
4
Encrypted text: lipps asvph


How it works is that each letter gets shifted by X amount of keys in the alphabet. 

So...

h (4 shifts) --> "l"
e --> "i"

vice versa.
